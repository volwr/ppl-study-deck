PPL Study Deck - Share Build

This is a share-safe static copy of the study app.

What is included:
- index.html
- styles.css
- app.js
- docs/chapters with focused FAA handbook chapter/excerpt PDFs
- External FAA, YouTube, and study resource links

What is intentionally not included:
- Saved progress
- localStorage persistence
- Personal data

To run:
1. Unzip this folder.
2. Open a terminal in the folder.
3. Run:
   python -m http.server 4175
4. Open:
   http://localhost:4175/

The lesson reader opens included chapter/excerpt PDFs. The source library and resource cards link to official FAA pages online.
